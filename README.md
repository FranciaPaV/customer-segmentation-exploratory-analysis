# Customer Segmentation and Exploratory Analysis

## Objective
Analyze customer behavior and segment users to support personalized marketing campaigns.

## Context
Building on a cleaned customer dataset, the business aims to better understand customer behavior and consumption patterns to improve marketing effectiveness.

## Data
Customer-level data including purchase history, spending patterns and demographic variables.

## Analysis
The project included:
- Exploratory data analysis (EDA)
- Customer segmentation based on behavioral patterns
- Identification of trends and high-value customer groups

## Tools
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Jupyter Notebook  

## Key Insights
- Clear behavioral differences exist between customer segments
- Segmentation enables more targeted and efficient marketing actions
- Data-driven insights support campaign personalization
